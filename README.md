# DrawIt!

_By: Abdullah Q._

## Overview:

This is a WIP project inspired by google's QuickDraw. For a while now I've been learning about AI and Machine Learning concepts, 
and taught myself PyTorch. I aim to utilize this foundation into creating a game that mimics QuickDraw.

At the moment, most of the UI is complete and the drawing logic is solid. Once I'm done the front-end, I'll start working on developing a 
pipeline between the canvas and a model. I want to create my own model and train it on the data that Google has provided for it's own game.
